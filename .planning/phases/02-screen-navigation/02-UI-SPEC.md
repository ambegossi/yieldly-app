## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `hsl(0 0% 100%)` — `background` CSS var | Screen background, card background |
| Secondary (30%) | `hsl(0 0% 96.1%)` — `muted` / `secondary` CSS var | "Current APY" label, muted text, badge background |
| Accent (10%) | `#00AD69` — `brand` Tailwind token | APY display value, chain badge green dot, token symbol icon bg, CTA button bg |
| Destructive | `hsl(0 84.2% 60.2%)` — `destructive` CSS var | Not used in this phase |

Accent (`brand` / `#00AD69`) reserved for:
1. APY value text (`text-brand`)
2. Green dot in chain badge (`text-brand`)
3. Token symbol icon container background (`bg-brand/10`) and border (`border-brand/20`)
4. CTA button background (`bg-brand` with white text)

No accent use on general interactive elements, borders, or non-data-highlight text.

Dark mode: all CSS variables shift per `global.css` dark block. Brand green stays `#00AD69` in both modes (no dark variant). CTA button uses explicit `bg-brand` class — no override needed.

Source: CONTEXT.md (green palette), `tailwind.config.js`, `global.css`, `src/screens/home/components/pool-list-item.tsx`.

## Screen Zones

Three visual zones from top to bottom (source: CONTEXT.md D-01, reference screenshots):

### Zone 1 — App Header
- Reuse existing `Header` component unchanged.
- Height: 64px + safe area inset top.
- Contains: Yieldly logo + "Yieldly" wordmark.

### Zone 2 — Navigation Link
- Full-width row below the header, `px-4 py-3`.
- Left arrow icon (`ChevronLeft` or `ArrowLeft` from lucide-react-native, 16x16, color `foreground`) + "Back to all coins" text (14px, weight 400, color `foreground`).
- Entire row is pressable (`Pressable` with `onPress={() => router.back()}`).
- Minimum touch target height: 44px.
- No visual card or border — plain pressable row on the background surface.

### Zone 3 — Token Identity Block
- Horizontal row: icon box on left, symbol text on right.
- Icon box: 56x56px, `rounded-2xl`, `bg-brand/10`, `border border-brand/20`. Contains symbol text (14px max, weight 600, color `brand`).
- Symbol text next to box: 28px, weight 700, color `foreground`.
- Token full name (e.g., "Tether"): rendered below the symbol at 14px, weight 400, color `muted-foreground`. NOTE: CONTEXT.md D-10 states the token full name is out of scope — the symbol field only. The reference screenshots show "Tether" as a subtitle but D-10 explicitly excludes it. **Omit the subtitle.** Only `pool.symbol` displays.
- Padding: `px-4 py-4` around the row.

### Zone 4 — Details Card
- White card (`bg-card`, `border border-border`, `rounded-2xl`, `shadow-sm shadow-black/5`).
- Margin: `mx-4` horizontal, `mt-2` top.
- Internal padding: `p-4`.
- Mobile layout (< 768dp): APY block stacked above project/chain block.
- Desktop layout (>= 768dp): APY block on the left, project/chain block on the right, in a single row.

**APY block:**
- Label: "Current APY", 14px, weight 400, color `muted-foreground`.
- Value: `{apy}%` formatted, 36px, weight 700, color `brand`. Use `formatAPY()` logic from PoolListItem.

**Project/chain block:**
- Project name: 18px, weight 600, color `foreground`.
- Chain badge: reuse `Badge` component (variant `subtle`). Contains green dot (• `text-brand`) + chain name text. Same pattern as PoolListItem.
- Mobile: project name then chain badge stacked vertically with 4px gap.
- Desktop: project name then chain badge stacked vertically, right-aligned within their column.

**Chart placeholder:** Phase 3 will insert the chart component into this card. Phase 2 renders no placeholder, no skeleton, no empty area. Source: CONTEXT.md D-03.

### Zone 5 — CTA Button
- Outside the card, below it.
- Margin: `mx-4 mt-6`.
- Mobile: full-width (`w-full`).
- Desktop: left-aligned, compact width (not full-width). Use `self-start` in the row.
- Button: use `Button` component with custom `bg-brand` className override (default variant uses `bg-primary` which is near-black — must override).
- Button content: "Open {project}" text + external link icon (`ExternalLink` from lucide-react-native, 16x16, color white) to the right of text.
- Text: 16px, weight 600, color white.
- Action: `Linking.openURL(pool.url)` or `expo-web-browser` `openBrowserAsync`.
- Source: CONTEXT.md D-08, D-09, ACTN-01, ACTN-02, reference screenshots.

## Accessibility Contract

- Back navigation link: `accessibilityRole="button"`, `accessibilityLabel="Navigate back to pool list"`
- APY value: `accessibilityLabel="Current APY: {formatted value}"`
- CTA button: `accessibilityRole="button"`, `accessibilityLabel="Open {project} in external browser"`

- Chain badge: no interactive role — decorative, read by screen reader as plain text
- Screen: wrapped in `ScreenWrapper` for Suspense + ErrorBoundary (consistent with home screen pattern)

    4. Replace `handlePoolPress` callback:

    FROM:
    ```typescript

      Alert.alert("Details coming soon", `${pool.symbol} on ${pool.project}`);
    }, []);
    ```

    TO:
    ```typescript
    const handlePoolPress = useCallback(
      (pool: Pool) => {
        router.push({
          pathname: "/pool-details",
          params: {
            id: pool.id,
            chain: pool.chain,
            project: pool.project,
            symbol: pool.symbol,
            apy: String(pool.apy),
            url: pool.url,
          },
        });
      },
      [router],
    );
    ```

    Key: `apy` must be serialized with `String(pool.apy)` because Expo Router params are strings. The `router` must be in the useCallback dependency array (Pitfall 6 from RESEARCH.md).

## Spacing Scale

Declared values (multiples of 4 only):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline spacing |
| sm | 8px | Compact element spacing, axis label gaps |
| md | 16px | Card internal padding, chart horizontal padding |
| lg | 24px | Section spacing within card |
| xl | 32px | Layout gaps between card sections |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Chart skeleton height: 200px (mobile) / 250px (desktop) — fixed pixel, not spacing scale token (source: CONTEXT.md D-06)
- Touch targets: minimum 44px per iOS HIG — applies to "Try again" retry button

## Typography

All text uses the existing `Text` component from `src/components/core/text.tsx`.

| Role | Size | Weight | Line Height | NativeWind Class | Usage |
|------|------|--------|-------------|------------------|-------|

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `hsl(0 0% 100%)` — white (`bg-card`) | Card background where chart renders |
| Secondary (30%) | `hsl(0 0% 96.1%)` — near-white (`bg-muted`) | Skeleton placeholder fill (pulsing gray rectangle) |
| Accent (10%) | `#00AD69` — brand green (`text-brand` / `bg-brand`) | Chart line only |
| Destructive | `hsl(0 84.2% 60.2%)` — red (`text-destructive`) | Not used in this phase |

Accent (`#00AD69`) reserved for:
- The Victory Native chart line stroke (source: CONTEXT.md D-05)

Muted foreground (`hsl(0 0% 45.1%)`) used for:
- Axis labels (Y-axis % values, X-axis date values)
- Caption text "APY history over the last 30 days"
- Error state body text "Failed to load chart"

Dark mode: all values follow existing CSS variable system in `global.css`. Chart line stays `#00AD69` in both modes (brand constant, not a semantic variable).

## Component Inventory

### New component: `ApyChart`

**Location:** `src/screens/pool-details/components/apy-chart.tsx`

**Responsibility:** Renders the Victory Native line chart with labeled axes. Handles all visual states internally.

**Props:**
```typescript
interface ApyChartProps {
  data: ApyDataPoint[];
  isPending: boolean;
  error: Error | null;
  onRetry: () => void;
}
```

**States and their visual treatment:**

| State | Visual |
|-------|--------|
| Loading (`isPending: true`) | Pulsing gray rectangle, 200px tall (mobile) / 250px tall (desktop), `bg-muted` fill, `rounded-lg`, `animate-pulse` via `tailwindcss-animate` |
| Error (`error !== null`) | Centered column: "Failed to load chart" (text-sm font-medium) + "Try again" Button (variant=outline, size=sm) |
| Empty (`data.length === 0`) | Centered "No data available" (text-sm text-muted-foreground) — no retry button |
| Success | Victory Native CartesianChart, green line (`#00AD69`), no area fill, labeled axes |

Caption "APY history over the last 30 days" renders in all four states below the chart area.

### Modified component: `PoolInfoCard`

**Location:** `src/screens/pool-details/components/pool-info-card.tsx`

**Change:** Accept `poolId: string` prop. Call `usePoolApyHistory(poolId)` internally. Render `<ApyChart>` below the existing APY/project info block, separated by `mt-6` (24px).
